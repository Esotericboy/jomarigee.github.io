    Question 7

    Read the file named 'ReadMeIfStuck.txt'. What is the Flag?

    Answer: Flag{******}

    First I opened the text file and was hoping it would be a simple flag sitting there that I could enter, but that sadly was not the case lol. When looking at the file it tells me two things, the first being that I'll have to look harder for the first flag. It gives me a file to search for, which I'll have to use the find command to find. The second thing that it tells me is what file to look for for a hint when going after the second flag.

    To find the file I entered the command find -type f -name additionalHINT. When looking at the file it tells me that I need to find a telephone directory.

    In order to find the telephone numbers directory I entered the command below which searches the root directory for the telephone numbers directory and redirects standard errors to /dev/null because it can be used to suppress output since it throws away any output its been given.

    ➜ find / -type d -name 'teleophone numebrs' 2>/dev/null

    Then I changed into the needed directories to find that the flag wasn't sitting in here, still more work to do lol. I need the use the find command and search for a modified file with the date of 2016-09-12 in the workflows directory.

    To find the file modified on the 12th I had to search within the 11th and 13th, which worked and it found the file that I needed. Then I transferred into that directory and looked at the file that I needed to see.

    ➜ find ~/workflows/ -type f -newermt 2016-09-11 ! -newermt 2016-09-13

    Once in the file I saw lots and lots of text, so I entered /Flag to find where the actual flag was in the text file. What a journey for one flag lol.

    Question 5

    Find a wordlist with the file extension of '.mnf' and use it to crack the hash with the filename hashC.txt. What is the password?

    Answer: u****************

    First, the wordlist needs to be found, so this is accomplished with the find command as pictured below

    ➜ find . -type f -name "*.mnf"

    Next, I need to find the hashC.txt file, so that it can be cracked. Which was done just like with the previous hashed text files

    ➜ find -type f -name hashC.txt

    Now I have to download both the wordlist and hashed password to my computer, so I can use John The Ripper to crack it.

    Then using hash-identifier I found out that its possibly a SHA-256 hash.

    Lastly using John The Ripper along with the password list provided I got the answer to the question. 

    ➜ sudo john --format=raw-SHA256 --wordlist=ww.mnf hashC.txt

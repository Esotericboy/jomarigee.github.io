    Question 2

    Copy this password hash to a file and research how to crack it. What is the cracked password?

    Answer: alqfna22

    First put those hashes that were just got onto your PC however you want. Since this is an old NLTM hash from Windows you would use the command below with a tool such as hashcat to crack the password. Set your hash to look like this below, so that you can crack it.

    Your passwords file should look like the one below, so that hashcat can actually run. This is done because

    The RID (1000) and LM Hash (aad3b435b51404eeaad3b435b51404ee) can be removed thus allowing the passwords to be cracked. Using the hashcat command below 

    ➜ hashcat -m 1000 -a 0 -o cracked.txt hash.txt ~/rockyou.txt

    Command Breakdown below -

    • ➜ -m: (hashtype) 1000 | NTLM
    • ➜ -a: (attack mode) 0
    • ➜ -o : (saving output) cracked.txt
    • ➜ hashfile : hash.txt
    • ➜ wordlist: rockyou.txt
